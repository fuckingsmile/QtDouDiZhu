\# Qt 联网斗地主项目



一个基于 Qt/C++ 开发的联网斗地主系统，包含客户端和服务端。



\## 项目结构

\- `client/`：客户端，负责界面、动画、出牌交互、网络通信

\- `server/`：服务端，负责房间管理、发牌、回合控制、胜负判定



\## 主要功能

\- 三人联网对战

\- 发牌、叫地主、出牌、Pass

\- 胜负结算

\- 基础动画和音效



\## 技术栈

\- C++

\- Qt Widgets

\- QTcpSocket / QTcpServer

\- JSON 协议通信



\## 运行方式



\### 服务端

1\. 打开 `server` 工程

2\. 编译并运行服务端



\### 客户端

1\. 打开 `client` 工程

2\. 编译并运行客户端

3\. 连接服务端 IP 和端口



\## 说明

本项目用于学习 Qt GUI、信号槽机制、TCP 网络通信以及基础游戏逻辑设计。
